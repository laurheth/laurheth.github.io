Pocket Dimensions RL

Play it on itch.io - GitHub

  • HTML5
  • CSS3
  • Javascript
  • Games
  • Game Jam Submission

Made in seven days for the 7DRL Challenge 2019. Pocket Dimensions is a roguelike set in a world which has been shattered into disjointed pockets of space. These pocket realities are connected by portals, which you have the ability to travel through and manipulate. Each pocket has a distinct biome, with differing denizens and climate. Every level is procedurally generated, presenting a different challenge on every playthrough, with the ultimate goal of finding the Wand of Nerual, held by the nefarious wizards who shattered the universe. Written in Javascript using the rot.js library.

Made in two weeks for Trans Gal Jam 3. Trains Girl Locomotion is a train building game about constructing a rail network to connect the many towns of Trains Girl Island. The citizens of those towns like to visit one another, and frequently organize parties, and the trains in your network automatically pick travellers up and drop them off at their destination. Snek, a Snake Game

View it on CodePen

  • HTML5
  • CSS3
  • ReactJS
  • Javascript

This is Snek, a game inspired by Nibbles, Snake, and similar. The player controls their snake, guiding it towards pellets of food. When food is eaten, the snake becomes longer. The player must avoid colliding with their own tail, or the walls of the play area. Before play begins, an AI snake plays the game as a demonstration. Written in Javascript using ReactJS.
